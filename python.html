Trail Twin


2/29/2024


Trail Twin is a social media app meant for connection people near you with similar outdoor activity interests. Users can Login/register, edit profile and upload profile photos. Also users can search for users by locations, activity interests, age and gender. Another feature is sending, accepting and declining friend. Friends can also create chats with each other. I used Python, Flask, WTForms, SQLAlchemy, Google Maps API, Cloudinary API, JavaScript, HTML, and CSS.
